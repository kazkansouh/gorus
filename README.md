# Gorus

A simple project in understanding how to use OpenGL from Go.

Draws a cube with hollow section.

```
      8                  9
       .-----------------.
      /                 /|
     /                 / |
   0/                1/  |
   .-----------------.   |
   |                 |   |
   |  .-----------.  |   |
   |  | 4/      5 |  |   |11
   |  | /         |  |   .
   |  |/6       7 |  |  /
   |  .-----------.  | /
   |                 |/
   .-----------------.
   2                 3
```
